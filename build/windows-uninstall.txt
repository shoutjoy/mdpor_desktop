Windows 제거 (dist에 생성되는 설치 파일)

[NSIS] *Setup*.exe 로 설치한 경우
  · 설정 → 앱 → 설치된 앱 → 해당 앱 → 제거
  · 또는 설치한 폴더의 Uninstall.exe 실행 (설치 시 복사됨)

[선택] dist에 MSI 설치 패키지도 만들려면 (제어판 제거 목록에 표시되기 쉬움)
  · PC에 WiX Toolset(candle, light)이 있어야 합니다.
  · package.json 의 build.win.target 배열에 { "target": "msi", "arch": ["x64"] } 를 추가하고
    build.msi 에 oneClick: false 등을 넣은 뒤 npm run build:win 을 다시 실행하세요.
