# setting
## 박중희 교수 
1. 앱을 설치한 후에 apikey와 인증번호를 입력해야 동작이 시작된다. 
인증번호는 SHA-256 해시로 검증하며, 평문은 소스/저장소에 저장하지 않는다. 
UI
[AI Studio API Key]
[ 입력란 ] 
       [API키 저장]

[인공지능 사용] [비번입력][저장]
ScholarAI [ ], sspimgAI [ ] 

2. [인공지능 사용] 에서 박스를 누르면, 인증번호 입력란이 나타난다. 
그리고 "박중희 교수님에게 인증번호를 받으세요"라고 나타나면서, 메일을 보낼 수있게 만든다. 이름, 학번, 전공, 연락처 를 입력받고 그 내용을  shoutjoy1@yonsei.ac.kr로 보내게 한다.  
3. 인공지능 사용[ ]설정에서 ScholarAI [ ], sspimgAI [ ] 에서 체크박스를 체크해야 인공지능 앱이 동작하게 된다. 이때부터 우측 사이드바에서 인공지능이 나타나다. 
물론 상단 메뉴에서 숨겨져있던 버튼도 나타난다.
