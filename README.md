# MDproViewer (Markdown & PPTX Viewer)

MDproViewer는 마크다운(Markdown) 문서와 PPTX(PowerPoint) 슬라이드를 고속으로 렌더링하고 유연하게 조작할 수 있는 전용 데스크톱 뷰어 애플리케이션입니다.

- **제작자**: 박중희 교수 (연세대학교 심리학과 겸임교수)

---

## 🛠️ 주요 기능

1. **마크다운 뷰어 & 에디터**
   - 실시간 프리뷰 및 특수 효과 지원
   - Mermaid 다이어그램 실시간 렌더링 및 Pan & Zoom, 마우스 드래그를 통한 컨테이너 크기 조절 기능 내장

2. **PPTX 고속 HTML 렌더러**
   - 썸네일 탐색 단계를 생략하고 직접 고속 HTML 렌더링 수행
   - **다양한 보기 모드 지원**:
     - **한 페이지**: 슬라이드 하나씩 전체 화면에 자동 맞춤 표시
     - **2 페이지**: 두 장의 슬라이드를 나란히 좌우 분할하여 표시
     - **매트릭스**: 화면 너비 및 확대/축소 비율에 따라 한 줄에 여러 슬라이드를 유동적으로 배치하는 반응형 뷰
   - **스케일링 및 자동 맞춤**: 창의 크기가 변경되거나 축소될 때 잘림 없이 슬라이드 전체가 비율에 맞게 축소(`Auto-Fit`)되어 렌더링됨
   - **네비게이션 연동**: 다중 페이지 보기 상태에서 원하는 슬라이드를 클릭하면 즉시 한 페이지 보기 모드로 전환되며 해당 페이지 위치로 이동

3.  **Electron 상단 메뉴 연동**:
    *   상단의 'View' 메뉴를 통해 한 페이지 스크롤, 두 페이지 보기, 매트릭스 보기를 직접 동적으로 선택 및 제어할 수 있습니다.

## 🚀 빌드 및 패키징 방법 (Windows)

이 프로그램은 Electron 프레임워크를 기반으로 하며, Windows 설치 파일(`.exe`)로 손쉽게 빌드할 수 있는 배치 스크립트를 제공합니다.

### 사전 준비 사항
1. **Node.js**가 시스템에 설치되어 있어야 합니다.
2. 프로젝트 루트 디렉토리(`변환자료`)에서 필요한 패키지들을 먼저 설치합니다:
   ```bash
   npm install
   ```

### 빌드 진행 단계
1. `C:\CusorApps\md-viewerVscode\변환자료` 디렉토리로 이동합니다.
2. 동봉된 **`build-win.bat`** 배치 파일을 실행합니다.
   - 더블 클릭하거나, 터미널에서 아래 명령어를 실행하십시오:
     ```powershell
     .\build-win.bat
     ```
3. 빌드가 시작되면 자동으로 Electron 소스가 컴파일되며, 배포 규격으로 패키징이 진행됩니다.

## 📦 설치 및 실행 방법

빌드가 성공적으로 완료되면 프로젝트 루트에 **`dist/`** 폴더가 생성됩니다.

### 설치 파일 위치
- **설치 프로그램 경로**: `C:\CusorApps\md-viewerVscode\변환자료\dist\`
- 해당 디렉토리 내에 생성되는 설치용 실행 파일(예: `MDproViewer Setup <버전>.exe`)을 실행하여 Windows 시스템에 설치를 진행해 주십시오.

### 실행 방법
- 설치가 완료되면 바탕화면 또는 시작 메뉴의 단축 아이콘을 통해 실행하거나, `dist/win-unpacked/MDproViewer.exe` 파일을 직접 열어 사용할 수 있습니다.

## 💡 유지보수 안내
- 본 PPTX 뷰어 및 마크다운 관련 렌더링의 핵심 웹 소스는 `web/pptx-viewer.html` 및 `web/js/` 디렉토리에 통합되어 관리되고 있습니다. 
- 경로 참조 및 Ajax 가로채기(CORS 방지 기법)가 정교하게 수립되어 있으므로, 신규 라이브러리 추가 시 절대 주소 대신 `./vendor/` 상대 주소 규칙을 지켜 유지보수 하시는 것을 권장합니다.
